# LMU Open Science Center Manual

This repository contains the [LMU Open Science Center (OSC)](https://www.osc.lmu.de/) lab manual, built as a [Quarto](https://quarto.org/) website. It serves as a central reference for OSC staff, members, and contributors, covering internal processes, best practices, and guidelines.

## Sections

- **Tech Overview** — High-level overview of our tech stack by proficiency level
- **About** — Overview of the OSC
- **Staff Onboarding** — Internal communication, working environment, and administration
- **OSC GitHub Organization** — Git workflows, repository management, GitHub Actions, Zenodo, and licensing
- **Updating the Website** — Creating news items, event pages, and member profiles
- **Onboarding an OSC Member** — Guide for onboarding new members
- **Organizing an OSC Event** — End-to-end event planning: program creation, registration, logistics, and feedback
- **OSC Public Outreach** — Social media, newsletters, and flyer design
- **Authoring an OSC Tutorial** — Tutorial template, Matomo analytics, versioning, and best practices
- **Switching a Group to Open Research Practices** — Surveys, templates, feedback, and iterative improvement
- **Matomo Analytics** — Overview, setup, campaigns, and adding sites
- **Archive** — Legacy documentation

## Tech Stack

- [Quarto](https://quarto.org/) for website generation
- HTML/CSS with [Cosmo theme](https://bootswatch.com/cosmo/)
- Hosted via GitHub Pages
- Matomo Analytics for usage tracking

## Local Development

To preview the site locally:

```bash
quarto preview
```

To render the site:

```bash
quarto render
```
