# LMU Open Science Center Manual

This repository contains the [LMU Open Science Center (OSC)](https://www.osc.lmu.de/) lab manual, built as a [Quarto](https://quarto.org/) website. It serves as a central reference for OSC staff, members, and contributors, covering internal processes, best practices, and guidelines.

## Contributions

This is a living and public document specific to the LMU Open Science Center, and as such, the primary contributors should be OSC staff members.

To future staff members: feel free to rearrange, edit, and update content as you see fit.

## Tech Stack

- [Quarto](https://quarto.org/) for website generation
- HTML/CSS with [Cosmo theme](https://bootswatch.com/cosmo/)
- Hosted via GitHub Pages

## Local Development

To preview the site locally:

```bash
git clone git@github.com:lmu-osc/manual.git
cd manual
quarto preview
```
