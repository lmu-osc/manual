# Wrap Up and Feedback

After an event, you should send emails to wrap up the event and also collect feedback.

## Wrap up

Send feedback form (formR) to all registrants and ask to fill within one week + advertise repository with event material (e.g. recordings, slides, etc)

Advertise repository with event material (e.g. recordings if gotten consent in advance, slides, etc) also on social media

Send feedback (formR report) to instructors

## Email Templates

**Mail to attendees**

Viele Grüße,
